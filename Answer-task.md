## Experiment 1

### 1. 为什么 Newton 方法在极小点附近比梯度下降快得多？

Newton 方法使用 Hessian 提供的二阶曲率信息。在一个 Hessian 非奇异的局部极小点附近，目标函数可以被一个二次模型很好地近似，而 Newton 步相当于直接求这个局部二次模型的极小点。在常见正则条件下，误差满足

```text
||x_{k+1} - x*|| = O(||x_k - x*||^2),
```

因此 Newton 方法具有局部二次收敛。

固定步长梯度下降只使用一阶梯度信息。它通常每次只把误差缩小一个固定比例，所以一般是线性收敛。正因为如此，当迭代点已经足够接近极小点时，Newton 方法会明显快于梯度下降。

在当前实验结果中，梯度下降用了 35 次迭代达到梯度容差，而 classical Newton 只用了 5 次迭代。

### 2. 为什么梯度下降在狭长等高线中经常出现 zigzag 行为？

在狭长等高线中，Hessian 的条件数较大，也就是说不同方向上的曲率差异很大。负梯度方向总是垂直于当前等高线，因此它常常指向山谷的横截面方向，而不是沿着山谷底部前进。若使用固定步长，迭代点很容易从山谷一侧跳到另一侧，于是路径表现出 zigzag 的来回震荡。

Newton 方法可以缓解这种现象，因为 Hessian 会根据局部曲率重新缩放梯度方向：在高曲率方向上有效步长较小，在低曲率方向上有效步长较大。

### 3. 数值结果是否支持 Newton 方法的 Q-quadratic convergence？

支持。当前结果清楚显示 Newton 方法具有非常快的局部收敛速度。Newton 误差和观测收敛阶为：

```text
k    error             q_k
0    1.4142135624      nan
1    0.7802557586      nan
2    0.3088151113      1.5585
3    0.0383447621      2.2507
4    8.988936e-05      2.9029
5    1.162104e-12      2.9994
```

本实验中观测阶并没有简单地趋近于 2，而是接近 3。这是因为测试函数关于 `x` 和 `y` 对称，并且初始点为 `(1, 1)^T`，所以 Newton 迭代始终保持在直线 `x = y` 上。在这条特殊对称轨道上，误差主导项表现得像更高阶项。

因此，更稳妥的结论是：该实验强烈支持 Newton 方法在极小点附近快速局部收敛；至于观测阶接近 3，则是本次对称初始条件和对称函数结构导致的特殊数值现象，而不是一般 Newton 方法的通用收敛阶。

## Experiment 2

### 1. 为什么 classical Newton 方法可能被 saddle point 吸引？

Classical Newton 方法本质上是在求解一阶驻点方程 `grad f(x) = 0`。它不会自动区分这个驻点是局部极小点、局部极大点还是鞍点。如果某个鞍点在 Newton 迭代映射下对当前初始点具有吸引性，那么迭代就可能收敛到该鞍点。

对于

```text
f2(x, y) = 0.5 x^2 + 0.25 y^4 - 0.5 y^2,
```

点 `(0, 0)` 是鞍点。从初始点 `(0.1, 0.1)^T` 出发，当前 classical Newton 实现收敛到近似点

```text
(0, -9.926167e-24),
```

最终函数值约为 `0`，最终梯度范数约为 `9.926167e-24`。这说明 classical Newton 确实可能收敛到鞍点。

### 2. Hessian indefinite 时，为什么 Newton direction 不一定是 descent direction？

Newton 方向为

```text
d = -H^{-1} g,
```

其中 `g = grad f(x)`，`H = Hessian f(x)`。一个方向是下降方向的条件是

```text
g^T d < 0。
```

如果 `H` 正定，那么 `H^{-1}` 也正定，因此对任意非零梯度 `g`，

```text
g^T d = -g^T H^{-1} g < 0。
```

但如果 `H` 不定，那么 `H^{-1}` 也不定。此时 `g^T H^{-1} g` 可能为正、为负或为零，所以 `g^T d` 不一定小于 0。因此，在 Hessian 不定时，Newton 方向不一定是下降方向。

### 3. 证明：若 `B_k` 正定且 `B_k d_k = -grad f(x_k)`，则当 `grad f(x_k) != 0` 时有 `grad f(x_k)^T d_k < 0`。

记

```text
g_k = grad f(x_k)。
```

由

```text
B_k d_k = -g_k
```

可得

```text
d_k = -B_k^{-1} g_k。
```

于是

```text
g_k^T d_k = -g_k^T B_k^{-1} g_k。
```

因为 `B_k` 正定，所以 `B_k^{-1}` 也正定。因此当 `g_k != 0` 时，

```text
g_k^T B_k^{-1} g_k > 0。
```

所以

```text
g_k^T d_k < 0。
```

这说明 `d_k` 是严格下降方向。

### 4. Armijo line search 在本实验中起什么作用？

Armijo line search 用来选择满足充分下降条件的步长。在 modified damped Newton 方法中，Hessian 修正先构造

```text
B_k = Hessian f(x_k) + tau_k I
```

使 `B_k` 正定，从而得到下降方向。随后 Armijo line search 控制沿该方向前进的距离，避免在非凸区域中盲目使用全步长导致函数值上升或跳到不稳定区域。

在当前实验中，这种组合避免了收敛到鞍点。Modified damped Newton 最终收敛到近似点

```text
(0, 1.0000000000230533),
```

最终函数值为 `-0.25`，最终梯度范数约为 `4.610667e-11`。

## Experiment 3

### 1. 为什么 `eta_k = 0.8` 的 Newton step 更便宜，但可能需要更多外层迭代？

Forcing term `eta_k` 控制内层 CG 方法求解 Newton 线性系统的精度。当 `eta_k = 0.8` 时，允许的残差较大：

```text
||H_k d_k + g_k|| <= 0.8 ||g_k||。
```

这意味着内层 CG 可以较早停止，所以单个外层 Newton 步的成本较低。但是，得到的方向只是对精确 Newton 方向的较粗略近似，因此外层迭代中梯度下降得较慢，可能需要更多外层迭代。

在当前实验结果中，`eta_k = 0.8` 需要 79 次外层迭代，总 CG 次数为 961。

### 2. 为什么 `eta_k = 0.1` 更准确但可能更昂贵？

当 `eta_k = 0.1` 时，内层 CG 必须把线性残差降得更低：

```text
||H_k d_k + g_k|| <= 0.1 ||g_k||。
```

因此计算出的方向更接近精确 Newton 方向，外层方法通常可以用更少的 Newton 迭代收敛。代价是每个外层步往往需要更多 CG 迭代。

在当前实验结果中，`eta_k = 0.1` 需要 16 次外层迭代，总 CG 次数为 861。

### 3. 为什么自适应规则 `eta_k = min(0.5, ||grad f(x_k)||)` 能平衡计算成本和收敛速度？

该自适应规则在远离最优点、梯度范数较大时使用较松的内层容差，从而避免在早期过度求解 Newton 线性系统，节省 CG 成本。

当迭代点接近最优点时，`||grad f(x_k)||` 变小，`eta_k` 也随之变小。此时内层求解更精确，有助于恢复 Newton 方法的快速局部收敛。

在当前实验结果中，自适应规则需要 15 次外层迭代，总 CG 次数为 1149。它的总 CG 成本高于两个固定规则，但最终梯度范数最小，约为 `8.207611e-13`。

### 4. 这个实验如何说明 inexact Newton 方法的基本思想？

该实验说明 Newton 方法并不总是需要精确求解 Newton 线性系统。可以用 CG 方法近似求解线性系统，并通过 forcing term `eta_k` 控制内层求解精度。

较大的 `eta_k` 会带来更便宜但较粗糙的方向；较小的 `eta_k` 会带来更准确但更昂贵的方向；自适应 `eta_k` 则根据当前迭代阶段动态调整精度，在远离解时节省成本，在接近解时提高精度。

这正是 inexact Newton 方法的核心思想：在内层线性代数成本和外层非线性收敛速度之间取得平衡。
