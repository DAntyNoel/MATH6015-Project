# MATH6015 Homework 7

This repository contains the source code, experiment scripts, generated figures, and written answers for the Homework 7 programming assignment on Newton-type optimization methods.

The implementation follows the suggested program structure in `homework7-programming.pdf`, Section 5. The main optimization algorithms are implemented directly in this repository. High-level optimization solvers such as `scipy.optimize.minimize`, MATLAB `fminunc`, or Julia `Optim.jl` are not used.

## Suggested Program Structure Mapping

The PDF suggests organizing the code into objective functions, optimization algorithms, line search, conjugate gradient, and a main experiment script. The repository maps those requirements as follows:

| PDF requirement | Repository location |
| --- | --- |
| Functions for evaluating `f(x)`, `grad f(x)`, and `Hessian f(x)` | `src/newton_methods/problems.py` |
| Gradient descent | `fixed_step_gradient_descent` in `src/newton_methods/algorithms.py` |
| Classical Newton method | `classical_newton` in `src/newton_methods/algorithms.py` |
| Modified damped Newton method | `modified_damped_newton` in `src/newton_methods/algorithms.py` |
| Armijo line search | `armijo_backtracking` in `src/newton_methods/line_search.py` |
| Conjugate gradient | `conjugate_gradient` in `src/newton_methods/algorithms.py` |
| Newton-CG / inexact Newton | `newton_cg` in `src/newton_methods/algorithms.py` |
| Main scripts that run experiments and generate figures | `exp1/run.py`, `exp2/run.py`, `exp3/run.py`, and root-level `run_all.py` |

## Environment

Install the Python dependencies before running the experiments:

```bash
pip install -r requirements.txt
```

The required third-party packages are listed in `requirements.txt`. The project uses NumPy for vector and matrix operations and Matplotlib for figure generation.

## Run Everything

From the repository root:

```bash
python run_all.py
```

This runs all three experiment scripts and regenerates the outputs under:

```text
exp1/outputs/
exp2/outputs/
exp3/outputs/
```

## Run Individual Experiments

Experiment 1 compares fixed-step gradient descent with classical Newton on

```text
f1(x, y) = x^4 + y^4 + x^2 + y^2 + 0.5xy.
```

Run it with:

```bash
python exp1/run.py
```

Experiment 2 compares classical Newton with modified damped Newton on the nonconvex function

```text
f2(x, y) = 0.5x^2 + 0.25y^4 - 0.5y^2.
```

Run it with:

```bash
python exp2/run.py
```

Experiment 3 compares Newton-CG forcing terms for

```text
f3(x) = 0.5 x^T A x + c/4 sum_i x_i^4.
```

Run it with:

```bash
python exp3/run.py
```
