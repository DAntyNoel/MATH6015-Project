# 实验 1：Newton 方法的快速局部收敛

本目录实现作业 PDF 中的 Experiment 1，用于比较固定步长梯度下降和经典 Newton 方法在函数

```text
f1(x, y) = x^4 + y^4 + x^2 + y^2 + 0.5xy
```

上的收敛行为。初始点为 `x0 = (1, 1)^T`，最优点为 `x* = (0, 0)^T`。

## 已实现内容

- 固定步长梯度下降，步长 `alpha = 0.2`
- 经典 Newton 方法
- 两种方法在 `f1` 等高线图上的迭代路径
- 误差 `||x_k - x*||_2` 关于迭代次数 `k` 的半对数图
- Newton 方法观测收敛阶 `q_k`

## 运行方式

在项目根目录运行：

```bash
python exp1/run.py
```

生成的图像和表格会写入 `exp1/outputs/`。

## 输出文件

- `experiment1_paths.png`：梯度下降与 Newton 方法的迭代路径等高线图
- `experiment1_errors.png`：两种方法的误差半对数曲线
- `newton_observed_orders.csv`：Newton 方法每一步的误差和观测收敛阶
- `summary.txt`：本次实验的数值摘要

## 关于观测收敛阶

本次运行中，Newton 方法最后一个有限的观测收敛阶约为

```text
q_k ≈ 2.9994
```

这比理论中常见的二阶收敛 `q_k -> 2` 更高。原因是本实验从对称初始点 `(1, 1)^T` 出发，而目标函数 `f1` 关于 `x` 和 `y` 对称，迭代会一直保持在直线 `x = y` 上。沿这条对称路径看，Newton 迭代的误差主导项表现得更像三阶项，因此数值上会观察到接近 3 的收敛阶。

这不意味着一般情形下经典 Newton 方法具有三阶收敛；更稳妥的结论是：该实验的对称结构让本次特定轨迹出现了超出通常二阶结论的特殊数值现象。写报告时可以指出这一点，并说明它仍然支持 Newton 方法在极小点附近具有非常快的局部收敛速度。
